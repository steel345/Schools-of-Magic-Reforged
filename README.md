# Schools of Magic

A port of the original Schools of Magic mod (Minecraft 1.12.2) to Minecraft 1.20.1 on Forge, with some extra content and reworks added along the way.

The original mod let a budding magician learn the secrets of magic with a wand and cauldron. This port keeps that and builds on it.

## Versions

- Minecraft 1.20.1
- Forge 47.3.0
- Mod id: `som`
